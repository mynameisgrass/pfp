## 💻 How to Use

### View Portfolio
```bash
# Already running! Just open:
http://localhost:3000
```

### Make Changes
Edit any component in `components/` folder and changes appear instantly!

```bash
# Example: Edit about section
# Edit components/About.tsx
# Save → Changes appear immediately in browser
```

### Add More Projects
Open `components/Projects.tsx` and add to the `projects` array:
```tsx
{
  title: 'Your New Project',
  description: 'What it does',
  image: '🎯',
  tech: ['Tech1', 'Tech2'],
  impact: 'What it achieved',
  status: 'Status',
}
```

### Customize Colors
Edit `tailwind.config.js`:
```js
colors: {
  primary: '#00D4FF',      // Change cyan
  secondary: '#FF006E',    // Change pink
  // ... etc
}
```
