### ✨ Visual Effects
- **Particle Animation**: Dynamic particle background with fluid motion and connection lines
- **Anime.js Animations**: Smooth, professional animations on scroll and page load
- **Gradient Text**: Beautiful gradient effects throughout the site
- **Neon Glow Effects**: Cyberpunk-inspired glowing elements
- **Smooth Scrolling**: Seamless navigation between sections
- **Responsive Design**: Mobile-friendly across all devices

## 🛠 Tech Stack

- **Frontend**: React 18, Next.js 14
- **Styling**: Tailwind CSS, Custom CSS animations
- **Animations**: Anime.js 3.2.1, Framer Motion
- **Build**: Next.js with TypeScript
- **Deployment**: Ready for Vercel, Netlify, or self-hosted

## 📦 Installation

```bash
# Clone the repository
cd F:/pfp

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm build

# Start production serve
npm start
```

## 🎨 Customization

### Colors
Edit `tailwind.config.js` to customize the color scheme:
- `primary`: #00D4FF (Cyan)
- `secondary`: #FF006E (Pink)
- `dark`: #0A0E27
- `darker`: #050811

### Content
All portfolio content is hardcoded in components for easy editing:
- `components/About.tsx` - About section content
- `components/Projects.tsx` - Project list (8 projects)
- `components/Skills.tsx` - Skills and expertise
- `components/Timeline.tsx` - Life timeline
- `components/Contact.tsx` - Contact information

### Discord/Social Links
Update the following in component files:
- `components/Header.tsx` - Discord profile link
- `components/Contact.tsx` - All social contact info
